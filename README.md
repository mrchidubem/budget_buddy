# Budget Buddy

## Project Overview

Budget Buddy is a personal finance management tool designed to help users track their income and expenses, establish budgets, and manage their financial goals effectively. Whether you're a student, a working professional, or anyone looking to take control of their finances, Budget Buddy provides a user-friendly interface and powerful features to assist you in your financial journey.

## Features
- Track income and expenses
- Set and manage budgets
- Generate financial reports
- User-friendly interface

## Getting Started

### Prerequisites
Before you start, ensure you have the following installed on your local machine:
- [Node.js](https://nodejs.org/) (version 14.x or later)
- npm (Node package manager)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mrchidubem/budget_buddy.git
   ```
2. Navigate to the project directory:
   ```bash
   cd budget_buddy
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Running the Project
To start the project, use the following command:
```bash
npm start
```
Open your browser and go to `http://localhost:3000` to view the application.

### Usage
- Create an account and log in.
- Start tracking your expenses and configuring your budget.
